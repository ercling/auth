Auth
=================================

Simple auth app.



Requirements
------------

The minimum requirement by this application that your Web server supports PHP 5.5.0.


Installation
------------

### Composer way: 
~~~
git clone https://github.com/ercling/auth.git
cd auth
composer install
~~~

Configure your server webroot to auth/web

Please check permissions on 
- auth/logs
- auth/data
- auth/runtime
